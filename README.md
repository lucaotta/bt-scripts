# bt-scripts
Development scripts for BTicino devices

# Usage
Use the provided install script
```
sh install.sh
```

# Features
* Automatically set the root folder in RW mode
* Provide useful shell aliases
  * t = `tail -f /var/log/BTicino.log`
  * tc = `touch /var/tmp/conf.xml`
  * ll = `ls -l`
  * la = `ls -a`
